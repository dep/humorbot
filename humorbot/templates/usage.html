Usage

Humorbot adds the /frink and /morbo commands to your Slack workspace. These search Frinkiac (The Simpsons) and Morbotron (Futurama) respectively.

Searches match against the show's subtitles. So searching for "Bender" may return a frame where another character says his name, rather than a frame showing Bender. Use /frink help or /morbo help to see quick usage info in Slack.

"I'm feeling lucky" mode

Just type a search query to get the first matching result with its subtitle as the text overlay:

/frink computer hacking
/morbo crushed pelvises

Custom text overlays

Separate your search query and custom overlay text with |:

/frink computer hacking | all this slack bot dev is making me thirsty
/morbo do the hustle | do the bartman

Use a trailing | with nothing after it to remove the overlay entirely:

/frink homer |

random mode

Returns a random frame from the search results, sent directly to the channel:

/frink random d'oh

images mode

Shows you a private selection of up to 10 matching frames, each with a Send button. Pick the one you like and it gets posted to the channel. Hit Cancel to dismiss without sending.

/morbo images seymour

gif mode

Generates an animated GIF from frames around the first search result (4 seconds on each side). You get a private preview with three options:

  • Send - post the GIF to the channel
  • Edit - open the frame editor to adjust start/end frames and toggle the text overlay
  • Cancel - dismiss without sending
/frink gif dental plan

In the editor, each frame shows a thumbnail with Start frame and End frame buttons. The selected range is marked with a green indicator. You can also toggle the text overlay on or off.

gifs mode

Like images but for GIFs - shows a selection of animated GIFs from different matching scenes:

/morbo gifs seymour
